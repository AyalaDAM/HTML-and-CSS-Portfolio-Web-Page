Last Projects

Rupee Clicker!


This project is a clicker game inspired by Cookie Clicker, where the player earns rupees by clicking, can buy items that automatically generate rupees, and unlock powers and achievements. It features a simple interface and dynamic functionality for an engaging and progressive experience.




The game is fully functional and offers a complete clicker experience, but it is not balanced in terms of difficulty and progression. All the key parameters—such as item costs and rupee generation rates are easily customizable within the JavaScript code. This flexibility allows you to fine-tune the gameplay to match your preferences, whether you want a more challenging experience or a faster progression. Additionally, the code is well-commented to help you understand where and how to make these adjustments, making it accessible for both beginners and more experienced developers who wish to experiment or expand on the project.



Technologies used:

  • Frontend:
    • HTML5 for structure.
    • CSS3 for styles, including some transitions.
    • JavaScript for game functionality.


Source code:

https://github.com/AyalaDAM/RupeeClicker

Landing Page


This project consists of a dynamic and professional landing page for a generic website. It features a simple and modern design, with interactive elements and very robust data validation on both the frontend and backend. It is also fully responsive and includes CSS effects, transitions via JavaScript, and form handling.




The landing page is fully functional if the necessary tools such as the database and the web server are managed. The design is implemented but not customized.



Technologies used:

  • Frontend:
    • HTML5 for structure.
    • CSS3 for styles, including transitions and blur effects.
    • JavaScript for form transitions, validations, and dynamic interactions.
  • Backend:
    • PHP for server-side validation, database interactions, and data processing.
  • Database: MySQL to store form data and dynamic content.

  • Additional tools:
    • JSON for data storage and management.
    • Private APIs (commented in the code).


Source code:

https://github.com/AyalaDAM/Landing-Page
